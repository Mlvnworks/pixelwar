# Pixelwar

Pixelwar is a small PHP codebase for a retro arcade light website and a gamified CSS game for students.

## Start Here

Project instructions are centralized in `development-guide/`.

Recommended reading order:

1. `development-guide/00_START_HERE.md`
2. `development-guide/CUSTOM_SETTING_PRIORITY.md`
3. `development-guide/01_SETUP_AND_ENVIRONMENT.md`
4. `development-guide/02_STRUCTURE_AND_RUNTIME.md`
5. `development-guide/03_EXTENSION_AND_EXAMPLES.md`

## Current Pages

- `/` - Landing page
- `/?c=home` - Learner home page
- `/?c=pixelwar` - `Pixelwar` interactive game test page
- `/?c=login` - Login page placeholder
- `/?c=signup` - Signup page placeholder

## Current Direction

- Tailwind-first styling
- Retro and arcade light theme
- Shared UI in `components/`
- Reusable logic in `classes/`
- Request handling in `submissions/`
